## CuraQとは？

**CuraQ** は、保存した記事をAIが自動で整理し、あなたに最適な記事を毎日お届けする知識キュレーションサービスです。

この拡張機能を使うと、Webで見つけた記事をワンクリックでCuraQに保存できます。

## この拡張機能でできること

- 右クリックメニューから記事を保存
- 拡張機能アイコンをクリックして記事を保存
- 保存前に内容を確認できる

## 必要なもの

- **CuraQアカウント**（無料で作成できます）: [CuraQ](https://curaq.app)
- **Google Chrome** ブラウザ

## インストール方法

Chrome Web Storeにはまだ公開されていないため、手動でインストールする必要があります。
難しそうに見えますが、手順通りに進めれば5分程度で完了します。

### ステップ1: 拡張機能をダウンロードする

1. このページの上部にある緑色の **「Code」** ボタンをクリックします
2. 表示されたメニューから **「Download ZIP」** をクリックします
3. ZIPファイルがダウンロードされます

### ステップ2: ZIPファイルを展開する

1. ダウンロードした **「curaq-chrome-extension-main.zip」** を見つけます
   - 通常は「ダウンロード」フォルダにあります
2. ZIPファイルをダブルクリックして展開します
   - Macの場合：ダブルクリックで自動展開
   - Windowsの場合：右クリック →「すべて展開」を選択
3. **「curaq-chrome-extension-main」** というフォルダができます

### ステップ3: Chromeに拡張機能を追加する

1. Chromeを開きます

2. アドレスバー（URLを入力する場所）に以下を入力してEnterキーを押します：
   ```
   chrome://extensions/
   ```

3. 画面右上にある **「デベロッパーモード」** のスイッチをONにします
   - 青色になればONです
   - ※「デベロッパーモード」とは、Chrome Web Store以外の拡張機能を追加できるようにする設定です

4. 画面左上に表示された **「パッケージ化されていない拡張機能を読み込む」** をクリックします

5. 先ほど展開した **「curaq-chrome-extension-main」** フォルダを選択します

6. 「CuraQ Saver」が一覧に表示されれば成功です

### ステップ4: 拡張機能をツールバーに固定する（推奨）

1. Chromeの右上にあるパズルピースのアイコン（拡張機能メニュー）をクリックします
2. 「CuraQ Saver」の横にあるピンアイコンをクリックします
3. ツールバーにCuraQのアイコンが常に表示されるようになります

## 使い方

### 方法1: 右クリックメニューから保存

1. 保存したい記事のページを開きます
2. ページ上で **右クリック** します
3. メニューから **「CuraQに保存」** を選択します
4. 確認画面が表示されるので、内容を確認して **「送信」** をクリックします

### 方法2: 拡張機能アイコンから保存

1. 保存したい記事のページを開きます
2. ツールバーの **CuraQアイコン** をクリックします
3. **「この記事を保存」** をクリックします
4. 確認画面で内容を確認して **「送信」** をクリックします

### プランによる違い

| 機能 | 無料/Standard | Pro |
|-----|--------------|-----|
| 右クリックで保存 | ○ | ○ |
| アイコンクリックで保存 | ○ | ○ |
| ポップアップ内で確認・保存 | - | ○ |

**Proプランの場合**、APIトークンを設定するとポップアップ内で直接保存できます。

<details>
<summary>APIトークンの設定方法（Proプランの方向け）</summary>

1. CuraQアイコンをクリックします
2. 歯車アイコン（設定）をクリックします
3. [CuraQ設定ページ](https://curaq.app/settings/access-token)でトークンを取得します
4. 取得したトークンを入力して保存します

</details>

## プライバシーについて

この拡張機能が送信するのは **URLとページタイトルのみ** です。

- ページの本文は送信されません
- ログイン情報やパスワードは送信されません
- CuraQのサーバーがURLから記事内容を取得します

## うまく動かないときは

### 「保存できない」場合

1. **CuraQにログインしていますか？**
   - [CuraQ](https://curaq.app) を開いてログイン状態を確認してください

2. **拡張機能が有効になっていますか？**
   - `chrome://extensions/` を開いて、「CuraQ Saver」がONになっているか確認してください

3. **ページを再読み込みしてみてください**
   - 拡張機能をインストールした直後は、ページの再読み込みが必要な場合があります

### 「デベロッパーモードの拡張機能を無効にする」という警告が出る場合

Chromeを起動するたびにこの警告が表示されることがあります。
**「拡張機能を有効にしたままにする」** をクリックすれば、引き続き使用できます。

※これはChrome Web Store以外からインストールした拡張機能に対して表示される標準的な警告です。

### それでも解決しない場合

[フィードバックページ](https://curaq.app/feedback)から状況をお知らせください。
できる限り対応いたします。

## アップデート方法

新しいバージョンがリリースされた場合：

1. このページから最新のZIPをダウンロードします
2. 古いフォルダを削除して、新しいフォルダに置き換えます
3. `chrome://extensions/` で「CuraQ Saver」の更新ボタン（丸い矢印）をクリックします
